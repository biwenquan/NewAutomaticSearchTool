1.Run "hight-zc.py"(need MILPSbox.pyc and on windows with python 3.*) to get 
"hight-line-17.lp"(The model file of 17 round zero-correlation of hight).

2.Run shell script "impossible.sh" to get the input and output pair of impossible trails
in "result". If the trail is impossible, it will print 'infeasible' just below the trail.

PS1:In "impossible.sh, you should set your gurobi.sh path to run on your linux system.
